# To-Do-List
To do list written in Typescript uses a Json database to store API inforamtion.
User can add tasks to do and set the staatus of them. The program will show how long until the task is due.
Colour of the table rows will change depending on the state of the task (completed-green, in-progress-orange, pending-yellow)
